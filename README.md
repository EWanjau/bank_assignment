# 💳 Lux Bank CLI Application

A simple command-line banking application built in Python using Object-Oriented Programming (OOP) principles. This app allows users to create an account, check balance, deposit money, and withdraw funds.

---

## Features

- Create a bank account with a default balance
- Check account balance
- Deposit money
- Withdraw money with validation
- Prevent negative transactions
- Prevent overdrawing (insufficient funds handling)
- Interactive CLI menu

---

## Concepts Used

- Python Classes & Objects
- Encapsulation
- Control Flow (if/elif/else)
- Loops (`while`)
- User Input Handling
